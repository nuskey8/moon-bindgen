# moon-bindgen
Automatically generates MoonBit FFI bindings to Rust/C libraries.

## 概要

moon-bindgenはMoonBit向けにRustやCライブラリのFFIコードを生成するためのライブラリです。RustのFFIコードを解析し、MoonBitのFFIコードを`build.rs`から自動で生成できます。[rust-bindgen](https://crates.io/crates/bindgen)と併用することにより、CライブラリのFFIコードを生成することも可能になっています。

moon-bindgenのコンセプトは[csbindgen](https://github.com/Cysharp/csbindgen)にインスパイアされています。

## 使い方

`cargo new`でパッケージを作成し、`[build-dependencies]`に`moon_bindgen`を追加します。

```bash
cargo add --build moon_bindgen
```

```toml
[package]
name = "example"
version = "0.1.0"

[lib]
crate-type = ["cdylib"]

[build-dependencies]
moon_bindgen = "1.0.0"

# optional (C FFI)
bindgen = "0.72" 
cc = "1.2.67"
```

## Rust to MoonBit

moon-bindgenはRustのFFIコードからMoonBitのFFIコードを生成できます。

```rust
// lib.rs

#[unsafe(no_mangle)]
pub extern "C" fn my_add(a: i32, b: i32) -> i32 {
    a + b
}
```

`build.rs`に以下のコードを追加します。

```rust
use std::error::Error;

fn main() -> Result<(), Box<dyn Error>> {
    moon_bindgen::Builder::default()
        .input_extern_file("src/lib.rs")
        .generate()?
        .write_moonbit_ffi_to_file("../ffi.mbt")?;
}
```

`cargo build`を実行すると以下のコードが生成されます。

```mbt
// ffi.mbt
// Generated by moon-bindgen. Do not edit.

extern "c" fn my_add(a : Int, b : Int) -> Int = "my_add"
```

> [!NOTE]
> ビルドした`.dll`、`.dylib`や`.so`などを利用する場合、linkerが適切にリンクできるよう`moon.pkg`から設定する必要があります。

## C to MoonBit

rust-bindgenと併用することでCのヘッダファイルからMoonBitのFFIコードを生成できます。

```h
// add.h
int add(int a, int b);
```

```rust
use std::error::Error;

fn main() -> Result<(), Box<dyn Error>> {
   bindgen::Builder::default()
        .header("includes/add.h")
        .generate()?
        .write_to_file("src/ffi.rs")?;

    let bindings = moon_bindgen::Builder::default()
        .input_bindgen_file("src/ffi.rs")
        .generate()?;
    bindings.write_to_file("../ffi.mbt", "../ffi_stub.c")?;

    Ok(())
}
```

## 所有権

MoonBitではFFIでリソースをやり取りする際、`#borrow`または`#owned`で引数の所有権を設定する必要があります。デフォルトでは全て`#borrow`になっていますが、`moonbit_ownership_resolver()`で関数の引数ごとに指定することが可能です。

```rust
moon_bindgen::Builder::default()
    .input_bindgen_file("src/ffi.rs")
    .moonbit_ownership_resolver(|func_name, param_name| {
        moon_bindgen::Ownership::Borrowed
    })
    .generate()?
    .write_moonbit_ffi_to_file("../ffi.mbt")?;
```

## Nullability

ポインタのnullabilityを`moonbit_nullability_resolver()`で変更できます。デフォルトでは戻り値のみがnullableで扱われます。

```rust
moon_bindgen::Builder::default()
    .input_bindgen_file("src/ffi.rs")
    .moonbit_nullability_resolver(|function, position| match (function, position) {
        ("context_get", moon_bindgen::NullabilityPosition::Return) => {
            moon_bindgen::Nullability::NonNull
        }
        ("context_set", moon_bindgen::NullabilityPosition::Parameter(name))
            if name == "context" => moon_bindgen::Nullability::Nullable,
        _ => moon_bindgen::Nullability::Unspecified,
    })
    .generate()?;
```

## 型のマーシャリング

| Rust                                      | MoonBit                    |
| ----------------------------------------- | -------------------------- |
| `()`                                      | `Unit`                     |
| `i8`, `u8`                                | `Byte`                     |
| `i16`, `u16`, `i32`                       | `Int`                      |
| `u32`                                     | `UInt`                     |
| `i64`, `isize`                            | `Int64`                    |
| `u64`, `usize`                            | `UInt64`                   |
| `f32`                                     | `Float`                    |
| `f64`                                     | `Double`                   |
| `bool`                                    | `Bool`                     |
| `c_char`, `c_schar`, `c_uchar`            | `Byte`                     |
| `c_short`, `c_ushort`, `c_int`            | `Int`                      |
| `c_uint`                                  | `UInt`                     |
| `c_long`, `c_longlong`                    | `Int64`                    |
| `c_ulong`, `c_ulonglong`, `size_t`        | `UInt64`                   |
| `c_float`                                 | `Float`                    |
| `c_double`                                | `Double`                   |
| `c_void`                                  | `Unit`                     |
| `*const c_char`, `*const i8`, `*const u8` | `ConstPtr[Byte]` / `Bytes` |
| `*mut c_char`, `*mut i8`, `*mut u8`       | `Ptr[Byte]` / `Bytes`      |
| `*mut *const T`                           | `Ref[ConstPtr[T]]`         |
| `*mut *mut T`                             | `Ref[Ptr[T]]`              |
| `*const c_void`, `*mut c_void`            | `RawPtr`                   |
| `*const T`, `*mut T`                      | `Ref[T]`, `#external` type |
| `extern "C" fn(...)`                      | `FuncRef`                  |
| `Option<extern "C" fn(...)>`              | `FuncRef?`                 |
| `#[repr(C)] struct`の値渡し               | MoonBit `struct` + C stub  |

`byte`ポインタはデフォルトで`Bytes`に特殊化されます。`moonbit_use_bytes(false)`を指定すると`Ptr[Byte]`または`ConstPtr[Byte]`として保持します。

多重ポインタは`Ptr[T]` / `ConstPtr[T]`で階層を保持します。

### C stub

MoonBitの通常のstructとC ABIのstructは同じレイアウトではありません。`#[repr(C)] struct`を値で受け取る、または返す関数に対しては、ABI変換用のC stubを生成します。

また、すべてのフィールドをマーシャリング可能な`#[repr(C)] struct`に対しては、`Type::new(...)`コンストラクタと`get_<field>()` / `set_<field>()`メソッドも生成します。

```rust
let bindings = moon_bindgen::Builder::default()
    .input_bindgen_file("src/ffi.rs")
    .moonbit_visibility(moon_bindgen::Visibility::Public)
    .c_stub_file_header("#include \"library.h\"")
    .c_stub_file_footer("// end of generated adapters")
    .generate()?;

bindings.write_to_file("../ffi.mbt", "../ffi_stub.c")?;
```

生成したstubは`moon.pkg`の`native-stub`へ追加します。

```moonbit
options(
  "native-stub": ["ffi_stub.c"],
  targets: { "ffi.mbt": ["native"] },
)
```

## Builderオプション

```rust
use heck::{ToShoutySnakeCase, ToSnakeCase, ToUpperCamelCase};
use std::error::Error;

fn main() -> Result<(), Box<dyn Error>> {
    let bindings = moon_bindgen::Builder::default()
        .input_bindgen_file("src/c_bindings.rs")
        .input_extern_file("src/lib.rs")
        .moonbit_file_header("")
        .moonbit_file_footer("")
        .c_stub_file_header("#include \"library.h\"")
        .c_stub_file_footer("// end of generated adapters")
        .moonbit_visibility(moon_bindgen::Visibility::Public)
        .function_filter(|name| !name.starts_with("_"))
        .type_filter(|name| !name.starts_with("_"))
        .constant_filter(|name| !name.starts_with("_"))
        .moonbit_function_rename(|name| name.to_snake_case())
        .moonbit_type_rename(|name| name.to_upper_camel_case())
        .moonbit_constant_rename(|name| name.to_shouty_snake_case())
        .moonbit_ownership_resolver(|function, parameter| moon_bindgen::Ownership::Borrowed)
        .generate()?;

    bindings.write_to_file("../ffi.mbt", "../ffi_stub.c")?;

    Ok(())
}
```
